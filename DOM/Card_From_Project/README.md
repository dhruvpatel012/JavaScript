# Dynamic User Profile Card Manager

A simple JavaScript project that allows users to create, edit, and delete profile cards dynamically using DOM manipulation.

## Live Demo

🔗 https://js-card-project.netlify.app/

---

## Features

* Create user profile cards dynamically
* Edit existing user profiles
* Delete user profiles
* Responsive UI for desktop, tablet, and mobile devices
* Modern card design with hover effects
* Dynamic DOM rendering
* Form validation for empty inputs

## How It Works

### Create Profile

1. Enter Name
2. Enter Email
3. Enter Image URL
4. Click **Create Profile**

A new user card will be generated instantly.

### Edit Profile

1. Click the **Edit** button on any card.
2. The form will be populated with existing user data.
3. Update the information.
4. Submit the form to save changes.

### Delete Profile

1. Click the **Delete** button.
2. The selected profile card will be removed from the UI.
